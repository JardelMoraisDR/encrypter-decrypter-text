Encrypt and Decrypt Text - Documentation

Introduction

Thank you for purchasing Encrypt and Decrypt Text. This tool is built with .NET Framework 4.8 and Windows Forms to provide a simple yet powerful way to encrypt and decrypt text or files using custom keys. This document will guide you on how to use and explore the source code of this product.

Package Contents

The product is delivered in a compressed ZIP file. Once extracted, the following directories will be available:

  • application - Contains the pre-built executable ready for use.
  • sourcecode - Contains the complete source code of the project.

Using the Application

To use the application, navigate to the application folder and run the executable file. No additional installation is required.

Running the Source Code

To run or modify the project using Visual Studio 2022, follow these steps:

  1. Open Visual Studio 2022.
  2. In the sourcecode folder, locate the file EncryptDecryptText.sln.
  3. Double-click the solution file to open it in Visual Studio.
  4. Build and run the project by selecting Build > Build Solution (or pressing Ctrl+Shift+B), then click Start to execute.
